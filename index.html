Session Protocol

  1. Read SOUL.md — identity
  2. Read USER.md — who to help
  3. Read memory/today.md + yesterday
  4. If main session → load MEMORY.md
  5. Check HEARTBEAT.md for tasks

Security Model

MEMORY.md only in main sessions (never in group chats). No credential exposure. No force push. No spending. Preview before external messages.
